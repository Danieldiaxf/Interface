Exercício 4 - HTML - Estrutura Geral

Bem - vindo

Um documento HTML é composto por tags estruturantes que permitem ao navegador interpretar e exibir informações diversas aos seus usuários

O documento HTMl é composto pelas seguintes tags estruturantes e por uma instrução, que são descritas a seguir:

Instrução DOCTYPE HTML

A instrução DOCTYPE HTML tem a função de dizer para o navegador qual é a versão do documento HTML que será aplicado ao documento. Essa instrução deve vir na primeira linha do documento. É fundamental para a renderização do documento HTML que será feito pelo navegador. Lembro que o DOCTYPE é uma instrução e não uma tag

TAG HTML

A tag HTML é uma tag estruturante, ou seja, é a tag raiz do documento html, essa tag envolve todo o documento estando presente no início e no fim do documento. Dentro dela teremos as outras tags estruturantes que são head e body, além de diversas outras tags que poderão estar nas tags filhas!

TAG HEAD

A tag HEAD é uma tag de cabeçalho. Os elementos HTML que estiverem dentro dela não aparecerão ao usuário e serão utilizadas para configurações utilizadas pelo navegador para exibir as informações visiveis. Essa TAG possuí apenas um único elemento visível que é a TAG TITLE

A TAG HEAD possuí tags de metadados sobre a página, codificação, links par acss e etc.

TAG BODY

A TAG BODY inclui todos os elementos que ficarão visiveis no documento HTML, tais como textos, mídias, arquivos de áudio, imagens e etc. Em síntese, tudo que estiver dentro do body estará visível para o usuário, salvo algumas exceções.
