Exercício 5 - HTML

Principais elementos

Os principais elementos html incluem tags para estrutura, texto. mídia e outros tipos de elementos que podem trazer mais interatividade com o documento. Essas tags são interpretadas pelos navegadores e então renderizadas e mostradas para os usuários

TAG p - Parágrafo

A TAG Paragrafo é utilizada para definir parágrafos. Neste bloco estamos usando-a. Essa tag vira dentro do sinal do diamante

TAG BR

A TAG BR é utilizada para quebra de linhas e podemos utiliza-la quantas vezes quisermos, nesse exemplo utilizaremos 3 quebras de linhas para o próximos exemplos.




TAG HR

A TAG hr é aquela que define / cria uma linha horizontal que possuí a largura do documento. Como exemplo vamos criar uma tag br a abixo!
