Elementos de Lista - Ul / Ol

Os principais elementos html para listas são o Ul para listas não ordenadas e o Ol para listas ordenadas. Esses elemntos são muito utilizados para exibir listas de itens de uma forma muito clara. São utilizados também para a criação de menus. A seguir escrevemos cada um deles.

Lista Não-Ordenada

A lista não ordenada é utilizada para agrupar itens onde a ordem é irrelevante, como por exemplo uma lista de compras. Os elementos dessa lista são bullets por padrão

Para representar uma lista usaremos a TAG ul ou ol além da TAG item. Abaixo exemplificamos o uso da lista não-ordenada - Exemplo:

  • Arroz
  • Feijão
    • kicaldo
      • salgado
      • caro
      • 500g
    • mamae
    • cristal
  • Batata
  • Bife
  • Coca
